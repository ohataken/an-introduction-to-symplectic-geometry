\subsection{ おおざっぱな定義 }

\begin{itemize}
  \item たくさんの集合からできている。それらの集合はあるルールを満たしていて、ひとつひとつを開集合と呼ぶ。
  \item それぞれの開集合は、集合からユークリッド空間 $ \mathbb{R}^n $ への関数 $ \varphi $ を持つ。
  \item 貼り合わせた部分がなめらかにつながる。
  \item シンプレクティック形式 $ \omega $ をもつ。
\end{itemize}
